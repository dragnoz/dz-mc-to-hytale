# DZ MC Models to Hytale Converter

An online platform for viewing and converting 3D block models between:
- **Minecraft Java** (`.json` block/item models)
- **Minecraft Bedrock** (`.geo.json` geometry)
- **Hytale** (`.blockymodel`)

## Goals

- View models in-browser with full 3D preview
- Convert between formats with one click
- Export Hytale-ready `.blockymodel` files for use in mods
- Eventually support custom block art for the DM topographic map system

## Status

Early development — viewer and format parsers in progress.
